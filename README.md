# Gym WhatsApp Bot & Dashboard

A full-stack monorepo for a Gym management system featuring a WhatsApp bot for members and a dashboard for owners.

## Tech Stack
- **Frontend**: React, TypeScript, Vite, TailwindCSS
- **Backend**: Node.js, Express, MongoDB, TypeScript
- **Integrations**: WhatsApp Cloud API, Razorpay, Google Gemini AI

## Folder Structure
- `client/`: React frontend (Vite)
- `server/`: Express backend
- `packages/types/`: Shared TypeScript definitions

## Getting Started

1. **Clone the repository**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Set up local environment**:
   Copy the root `.env.example` to `server/.env` and `client/.env` and fill in the secrets.
4. **Run in development mode**:
   ```bash
   npm run dev
   ```
